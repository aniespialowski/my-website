O mnie

Tematyka kodowania zainteresowała mnie stosunkowo niedawno, mniej więcej na przełomie 2019/2020, kiedy zacząłem pracę w rekrutacji IT. Do moich codziennych obowiązków wówczas należało pozyskiwanie dla naszych partnerów biznesowych pracowników z różnych obszarów sektora IT. W toku poszukiwań od Testerów, przez Storage Engineerów, Big Data Developerów,
DevOps-ów, Java Developerów itd. trafiłem wreszcie na profil Frontend Developera, którego obszar pracy tak mnie zainteresował, że postanowiłem sam rozpocząć naukę w tym obszarze. Poza forami dyskusyjnymi, artykułami oraz różnymi tutorialami w sieci - zrealizowałem szereg kursów internetowych (głównie Udemy oraz StrefaKursów.pl) które poszerzyły i uporządkowały moją dotychczasową wiedzę oraz ukierunkowały moje oczekiwania w kontekście obszarów do dalszego rozwoju.

Za pomocą tych platform do tej pory zrealizowałem m.in takie kursy jak (łącznie ponad 120 godzin):

➔  Kurs HTML 5.1 - podstawy tworzenia stron
➔  Kurs CSS3 - kodowanie ze stylem
➔  Produktywne tworzenie stron z LESS & SASS
➔  Flexbox & Grid - responsywne layouty od podstaw
➔  Kurs HTML i JavaScript - praktyczne projekty
➔  Kurs Bootstrap 4
➔  Jak przyspieszać i optymalizować strony
➔  Wprowadzenie do tworzenia aplikacji mobilnych
➔  The Web Developer Bootcamp 2020

Obecnie pracuję jako Freelancer - współpracuję głównie z firmą Startup-ową, z którą budujemy sklepy internetowe głównie dla klientów z Europy Zachodniej. Nasze działania opieramy przede wszystkim na rozwiązaniach kanadyjskiej platformy Ecommercowej "Shopify".

Z chęcią wspomogę również Państwa w zbudowaniu Państwa sklepu czy strony internetowej.

Zachęcam do kontaktu !

Umiejętności

HTML5

Dobra znajomość, umożliwiająca mi codzienną pracę zgodną ze standardami W3C.

CSS3

Dobra znajomość. Staram się, żeby każdy projekt był zgodny z ideą "Responsive Web Design".

LESS

Prosty i wygodny w obsłudze preprocesor CSS. Dobrze się sprawdza przy większych projektach.

SASS

Ciekawa alternatywa dla LESS. Trochę bardziej skomplikowany, ale oferujący za to więcej możliwości.

JavaScript

Najciekawsze narzędzie do budowy frontendu. Dzięki JS można wzbogacić stronę o interesujące efekty, animacje itp.

GitHub/GIT

Kontrola wersji kodu to niezbędnik każdego Developera. Github z kolei jest świetną bazą do przechowywania repozytoriów.

Bootstrap 4/5

Przydatny framework CSS z wbudowaną siatką Grid. Przydatny przy złożonych projektach.

Shopify & Liquid

Ciekawa alternatywa dla klasycznych platform CMS. Shopify to dobra opcja do rozwoju biznesu internetowego.
