O mnie

Tematyka kodowania zainteresowała mnie stosunkowo niedawno - mniej więcej wtedy, gdy rozpocząłem współpracę z obecnym pracodawcą. Do moich codziennych obowiązków należało pozyskiwanie dla naszych partnerów biznesowych pracowników z różnych obszarów sektora IT. W toku poszukiwań od Testerów, przez Storage Engineerów, Big Data Developerów,
DevOps-ów, Java Developerów itd. trafiłem wreszcie na profil Frontend Developera - najpierw ze znajomością Angular później z React. Obszar pracy Frontend Developera tak mnie zainteresował, że rozpocząłem samodzielną naukę w tym obszarze. Poza forami dyskusyjnymi, artykułami oraz różnymi tutorialami w sieci - realizuję konsekwentnie kursy internetowe, które poszerzają oraz uporządkowują moją dotychczasową wiedzę. Za pomocą platform: StrefaKursów.pl oraz Udemy do tej pory zrealizowałem następujące kursy (łącznie ponad 70h):

- Wprowadzenie do tworzenia stron internetowych
- Kurs HTML 5.1 - podstawy tworzenia stron
- Kurs CSS3 - kodowanie ze stylem
- Produktywne tworzenie stron z LESS & SASS
- Flexbox & Grid - responsywne layouty od podstaw
- Warsztaty tworzenia stron internetowych
- Kurs HTML i JavaScript - praktyczne projekty
- The Web Developer Bootcamp 2020

Obecnie szukam pracy, która umożliwi mi zdobycie pierwszego doświadczenia komercyjnego, aby w niedalekiej przyszłości zająć stanowisko Frontend Developera z React.js.

LESS
